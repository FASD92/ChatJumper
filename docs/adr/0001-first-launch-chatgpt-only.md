# First launch supports ChatGPT only

ChatJumper will launch first with ChatGPT as the only supported site, while Gemini and Claude are described as coming soon rather than supported. The product specification originally described ChatGPT, Gemini, and Claude as the MVP target, but narrowing the first launch reduces host-match surface, selector drift risk, screenshot burden, and live smoke-test scope while still validating the core value of jumping back to the latest user question in a long AI chat.
