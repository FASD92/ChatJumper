개인정보 지원 English GitHub

Privacy-first Chrome extension

최신 ChatGPT 질문으로 다시 이동하세요.

ChatJumper는 긴 ChatGPT 대화에서 Composer Button, 키보드 단축키, highlight feedback으로 사용자 질문 위치를 빠르게 되찾게 해 줍니다.

Chrome Web Store 공개 예정 소스 보기

하나의 일에 집중합니다

Composer Button

입력창 근처의 ChatJumper 버튼으로 사용자 질문 사이를 이동합니다.

Keyboard Shortcut

Windows/Linux에서는 Alt+J, macOS에서는 Command+J로 같은 동작을 실행합니다.

Highlight Feedback

점프한 뒤 목표 질문을 강조해 현재 위치를 빠르게 파악할 수 있습니다.

동작 방식

ChatJumper는 브라우저의 현재 페이지 DOM을 읽어 열린 ChatGPT 대화의 사용자 질문 위치를 찾고 로컬로 스크롤합니다. 검색, 채팅 내보내기, 서버 전송 기능은 제공하지 않습니다.

Privacy-first 기본값

채팅 저장 없음

ChatJumper는 채팅 원문을 저장하지 않습니다.

서버 전송 없음

대화 내용은 외부 서버로 전송하지 않습니다.

Analytics 없음

첫 출시는 analytics나 telemetry를 포함하지 않습니다.

지원 사이트

첫 출시는 ChatGPT를 지원합니다. Gemini와 Claude는 지원 예정입니다.
