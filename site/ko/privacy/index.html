개인정보 처리방침

ChatJumper는 현재 ChatGPT 페이지 안에서 사용자 질문 위치로 이동하는 local-first Chrome 확장입니다.

데이터 처리

ChatJumper는 채팅 원문을 저장하지 않습니다.

ChatJumper는 채팅 원문을 외부 서버로 전송하지 않습니다.

ChatJumper는 analytics를 사용하지 않습니다.

ChatJumper는 사용자 설정만 chrome.storage.local에 저장합니다.
