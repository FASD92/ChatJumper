Privacy Policy

ChatJumper is a local-first Chrome extension for jumping to user questions inside the current ChatGPT page.

Data handling

ChatJumper does not store chat content.

ChatJumper does not send chat content to any server.

ChatJumper does not use analytics.

ChatJumper stores user preferences only in chrome.storage.local.
