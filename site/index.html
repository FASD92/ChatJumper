Jump back to your latest ChatGPT question.

ChatJumper keeps long ChatGPT conversations navigable with a floating Composer Button, keyboard shortcut, and visible highlight feedback.

Coming soon on Chrome Web Store View source

Built for one job

Composer Button

Click the ChatJumper button near the composer to move through your user questions.

Keyboard Shortcut

Use Alt+J on Windows/Linux or Command+J on macOS for the same jump behavior.

Highlight Feedback

After a jump, ChatJumper highlights the target question so you can reorient quickly.

How it works

ChatJumper reads the current page DOM in your browser, identifies user-question positions in the open ChatGPT conversation, and scrolls locally. It does not replace search, export chats, or send content to a server.

Privacy-first by default

No chat storage

ChatJumper does not store chat content.

No server transfer

Your conversation content is not sent to any server.

No analytics

The first launch does not include analytics or telemetry.

Supported site

ChatGPT is supported for the first launch. Gemini and Claude are coming soon.
